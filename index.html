EZ Miner

Start mining Ethereum in 5 easy steps!

(EZ Miner is currently only available for Windows)

Step 1: Download an Ethereum wallet so you can get paid. We recommend the Exodus Wallet. If you already have an Ethereum wallet, skip this step. Download Exodus

Step 2: Download EZ Miner App

(Remember where you download the zip file to)

Step 3: Extract the EZMiner.zip file by Right-Clicking on it and choosing "Extract All..."

Step 4: Configure Miner

In the EZ Miner App, click Configure > Configure EZ Miner. Fill in your Ethereum address and other information, then click the Save Configuration button on the bottom.

Step 5: Start mining!
